# Update AGENTS.md
​
Review the conversation for any durable, generalizable lesson about working in this repo.
​
## Rules
​
- **Only update if** a durable lesson was learned: a principle, process, debugging heuristic, or coding convention applicable beyond this specific task.
- **Skip if** the lesson cannot be stated without referencing a specific file, selector, or component—unless it reflects a broader rule.
- **One short bullet** under an appropriate existing section. Add a new section only if absolutely necessary.
- **If nothing durable was learned, make no changes.**
​
## Process
​
1. Review the conversation for any generalizable insight.
2. If found, locate the relevant section in AGENTS.md in the repo root.
3. Append one concise bullet. Do not reformat or restructure existing content.
4. If AGENTS.md doesn't exist and the lesson warrants it, create a minimal one with just the new entry.
